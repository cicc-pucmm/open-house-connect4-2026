<h1 align="center">OpenHouse 2026 Social App</h1>

<p align="center">
  Connect4 para el evento OpenHouse 2026. Permite a los asistentes jugar contra la máquina u otros jugadores utilizando una aplicación de consola elaborada en C.
</p>

## Requisitos Previos

- Sistema Operativo de Windows
- Compilador de C (Para compilar el source file)
- Librería(s) no estándar(es) conio.c, conio.h

## Ejecución
- Ejecutar archivo (.exe)
- ALTERNATIVA: Compilar el código.

## Funcionalidades

### Juego Unijugador
- Elección de color de jugador (Color aleatorio para el computador).
- Gameloop contra el computador hasta llenar el tablero (empate) o victoria de alguno de los partidos.

### Juego Multijugador (2 Jugadores
- Elección de color de cada jugador.
- Gameloop entre jugadores hasta llenar el tablero (empate) o victoria de alguno de los partidos.
